a_e_xiii_library_1.0.1.php
==============================

This is php functions created using php, html, css and js to make web development easy and faster;
It uses only php functions which takes in an array argument or sometimes a function as an argument.


HOW TO INCLUDE THE LIBRARY
============================
<?php
    include 'path/a_e_xiii_library_1.0.1.php';
?>

==========================
